#  Flappy Ball Webgame

A lightweight, responsive, and addictive physics-based web game. This is a modern twist on the classic "flap" mechanic, replacing the bird with a dynamic ball, built entirely with **HTML5, CSS3, and Vanilla JavaScript**.

##  Gameplay
The objective is simple: navigate the **ball** through a series of obstacles without making contact. Each gap you pass successfully increases your score!

### **Controls**
- **Spacebar / Click / Tap:** Defy gravity and bounce the ball upward.
- **Goal:** Navigate as many obstacles as possible to set a new high score.

---

##  Features
- **Physics-Based Movement:** Smooth gravity and velocity handling for the ball.
- **Responsive Design:** Playable on desktops, tablets, and mobile browsers.
- **High Score Tracking:** Keeps track of your best run during the session.
- **Clean UI:** Minimalist aesthetic focused on smooth 60 FPS gameplay.
- **Collision Detection:** Precise hitbox logic for fair and challenging play.

---

##  Built With
- **HTML5:** Game structure and canvas rendering.
- **CSS3:** Layout styling and responsive container management.
- **JavaScript (ES6+):** Core game engine, gravity physics, and collision logic.

## Installation & Setup

Navigate to the folder:

```Bash
cd Flappy-Bird-Webgame
```
Launch the game:
Simply open index.html in any modern web browser.
