## üéÆ Gameplay
The objective is simple: navigate the **ball** through a series of obstacles without making contact. Each gap you pass successfully increases your score!

### **Controls**
- **Spacebar / Click / Tap:** Defy gravity and bounce the ball upward.
- **Goal:** Navigate as many obstacles as possible to set a new high score.

## üöÄ Features

- **Responsive Design:** Playable on desktops, tablets, and mobile browsers.

- **Clean UI:** Minimalist aesthetic focused on smooth 60 FPS gameplay.
